## 🧠 **4. Thought Process**

### **思考步驟：**

### ✅ **堆疊法 (Stack Approach)**  
1. **遇到開括號 `(`, `{`, `[` 時，將其推入堆疊。**
2. **遇到關括號 `)`, `}`, `]` 時，檢查堆疊：**
   - 如果堆疊為空，直接返回 `False`。
   - 如果堆疊頂部是對應的開括號，則將其**彈出**。
   - 如果堆疊頂部不匹配，返回 `False`。
3. **最終檢查：**
   - 如果處理完所有字符後，堆疊為空，則返回 `True`。
   - 如果堆疊不為空，則返回 `False`。
